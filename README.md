# Image Slider

A clean, lightweight image slider built with HTML, CSS, and JavaScript. The project includes automatic slide rotation, manual navigation controls, and a simple responsive layout designed for quick integration into small websites or demos.

## Features

- Automatic slideshow playback
- Previous and next navigation buttons
- Simple, minimal implementation
- Easy to customize with your own images
- Lightweight front-end only project

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```text
index.html
style.css
script.js
image1.jpg
image2.jpg
image3.jpg
```

## How It Works

- `index.html` defines the slider structure and loads the stylesheet and script.
- `style.css` controls the image visibility and basic presentation.
- `script.js` handles slide switching, autoplay, and navigation button behavior.

## Getting Started

### 1. Clone or download the project

Place the files in a local folder and make sure the image files are in the same directory as `index.html`.

### 2. Open the project

Open `index.html` in a browser to run the slider.

### 3. Customize the images

Replace `image1.jpg`, `image2.jpg`, and `image3.jpg` with your own images, or update the file names in `index.html` if needed.

## Usage Notes

- The slider starts automatically when the page loads.
- Use the left and right buttons to move between slides manually.
- If you add or remove images, update the HTML to match the number of slides.
